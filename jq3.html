JQuery Selectors

Jquery use css like selectors. But Unlike css, jquery is having many more selectors than CSS2. Here is a list of complete jquery selectors with live example and code

Jquery code Description
$("element") $("p").css({ "color":"red"}) select all elements with tag name in example.
$("#id") $("#p1").css({ "color":"blue"}) select single element with id name in example. If id name is repeated, only first match will be executed
$("ul,ol") $("ul,ol").css({ "color":"gray"}) select grouping of ul and ol element.
$("ul > li") $("ul > li").css({ "color":"aqua"}) select all child selectors of ul element

Jquery Basic Filters

JQuery Code Descrption
$(":first") $("ul li:first").css({ "background":"red"}) select the first match elementin example.
$(":last") $("ul li:last").css({ "background":"red"}) select the last match elementin example
$(":gt(index)") $("ul li:gt(2)").css({ "background":"red"}) select elements greater than nth index in example.
$(":eq(index)") $("ul li:lt(2)").css({ "background":"red"}) select elements greater than nth index in example.
$(":odd") $("ul li:odd").css({ "background":"red"}) select all odd indexed elements in example

Filter by Content

Jquery Code Description
$(":contains(text)") $("p:contains(hello)").css({ "background":"red"}) select the p element with text "hello"in example.
$(":has(selector)") $("ul li:has(a)").css({ "background":"red"}) select only elements with descendants a in example.
$(":parent") $("table tr td:parent").css({ "background":"red"}) :parent is inverse of : empty in example.

Attribute Filters

JQuery Code Description
$("[ attribute ]") $("ul li[id]").css({ "background":"red"}) select element with attribue name
$("[ attribute!='value' ]") $("li[ class!='col']").css({ "outline":"solid red"}) select element with attribute value not equal to value
$("[ attribute='value' ]") $("input[ name='firstname']").css({ "outline":"solid red"}) select element with attribute name and value
$("[ attribute$='value' ]") $("li[class$='2']").css({ "outline":"solid red"}) select element with attribute value ending exactly with given string
