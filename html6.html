HyperLink

HTML Hyperlinks are created using <a> tag. A hyperlink links a webpage with other webpages and external pages. A hyperlink can be Internal Link, External Link, Internal Link, Mail Link or telephone link.

In HTML5, an hyperlink can contain both block level and inline level elements. That means, we can write text, image or a div inside hyperlink.
